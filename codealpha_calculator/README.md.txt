# 🎵 Music Player

A sleek and responsive web-based **Music Player** built using **HTML, CSS, and JavaScript**. It provides a modern UI with album artwork, playback controls, playlist support, progress tracking, and volume control.

---

## ✨ Features

* 🎶 Play, Pause, Next & Previous controls
* 📂 Playlist overlay
* 🖼️ Dynamic album artwork
* ⏱️ Real-time progress bar
* 🔊 Volume control
* 🔁 Automatically plays the next song
* 📱 Responsive and modern UI
* 🎵 Displays song title and artist

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/music-player.git
```

2. Navigate to the project folder

```bash
cd music-player
```

3. Open **index.html** in your browser.

No additional installation or dependencies are required.

## 🎮 Controls

| Action           | Description                       |
| ---------------- | --------------------------------- |
| ▶️ Play/Pause    | Starts or pauses the current song |
| ⏮ Previous       | Plays the previous song           |
| ⏭ Next           | Plays the next song               |
| 🎵 Playlist      | Opens the playlist panel          |
| 📍 Progress Bar  | Seek to any part of the song      |
| 🔊 Volume Slider | Adjust playback volume            |
